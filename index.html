HTML Commands

  • <!DOCTYPE html> - Declares the document type and HTML version.
  • <html> ... </html> - Root element that contains the entire HTML document.
  • <head> ... </head> - Contains metadata and links to scripts/styles.
  • <title> ... </title> - Sets the page title shown in the browser tab.
  • <body> ... </body> - Contains all visible webpage content.
  • <h1> ... </h1> - Heading level 1, largest heading.
  • <p> ... </p> - Defines a paragraph.
  • <a href="URL"> ... </a> - Creates a hyperlink.
  • <img src="image.jpg" alt="desc"> - Inserts an image.
  • <ul> ... </ul> - Unordered (bulleted) list.
  • <ol> ... </ol> - Ordered (numbered) list.
  • <li> ... </li> - List item.
  • <div> ... </div> - Generic container element.
  • <span> ... </span> - Inline container for styling text.
  • <form> ... </form> - Defines a form for user input.
  • <input type="text"> - Text input field.
  • <button> ... </button> - Clickable button.
  • <table> ... </table> - Table structure.
  • <tr> ... </tr> - Table row.
  • <td> ... </td> - Table cell.

  • ul { list-style-type: square; } - Square bullet points.
  • body { font-family: Arial, sans-serif; } - Font family.
  • header { position: fixed; top: 0; } - Fixed header at top.
  • .container { max-width: 1200px; margin: auto; } - Center container.
  • @media (max-width: 600px) { ... } - Responsive design query.
  • button { cursor: pointer; } - Pointer cursor on buttons.
  • img { border-radius: 50%; } - Circular images.
  • input:focus { outline-color: blue; } - Outline color on input focus.
  • body { margin: 0; } - Remove body margin.
  • p::first-letter { font-size: 200%; } - Style first letter.
  • nav { background-color: #333; } - Nav background color.

PyGame Commands

  • import pygame - Import PyGame module.
  • pygame.init() - Initialize PyGame.
  • screen = pygame.display.set_mode((800, 600)) - Set screen size.
  • pygame.display.set_caption("Game") - Set window title.
  • clock = pygame.time.Clock() - Control frame rate.
  • running = True - Game loop control variable.
  • for event in pygame.event.get(): - Get input events.
  • if event.type == pygame.QUIT: - Quit event check.
  • screen.fill((0, 0, 0)) - Fill screen with black.
  • pygame.display.flip() - Update display.
  • clock.tick(60) - Limit to 60 FPS.
  • pygame.draw.rect(screen, (255, 0, 0), (100, 100, 50, 50)) - Draw rectangle.
  • pygame.draw.circle(screen, (0, 255, 0), (200, 200), 40) - Draw circle.
  • pygame.image.load("player.png") - Load image.
  • screen.blit(image, (0, 0)) - Draw image to screen.
  • pygame.mixer.Sound("sound.wav").play() - Play sound.
  • pygame.key.get_pressed() - Get key states.
  • pygame.font.SysFont(None, 36) - Load font.
  • font.render("Text", True, (255, 255, 255)) - Render text.
  • pygame.quit() - Quit PyGame.
